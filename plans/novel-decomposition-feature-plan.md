# 小说分解附加功能实施计划

## 目标

为同人模式新增“小说分解附加功能”，支持长篇小说分段拆分、结构化总结、树状注入、断点续跑、后台拆分、JSON 分享与运行时注入查看。

## 总体范围

- 首页新增小说分解独立工作台入口
- 设置内仅保留小说分解独立 API 配置
- 支持独立 API 设置
- 支持分段拆分、进度保存、断点恢复
- 支持后台拆分与边玩边拆
- 支持主剧情/规划分析/世界演变三层注入
- 支持树状多条目管理与注入查看
- 支持 JSON 导入导出与分享

## 关键设计原则

- 主剧情只吃轻量概括，不直接吞详细树
- 已完成分段立即可注入，未完成分段不阻塞游玩
- 所有长任务必须可恢复，避免一次失败全盘重来
- 结构化结果优先，原始文本只做辅助展示
