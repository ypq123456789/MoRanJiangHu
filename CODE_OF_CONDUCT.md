# 行为准则

本项目希望成为一个适合长期协作、乐于交流、愿意共同打磨作品的开源空间。

无论你是维护者、首次贡献者还是旁观者，在本仓库、Issue、Pull Request、Discussion 以及其他相关沟通场景中，都请遵守以下准则。

## 我们鼓励的行为

- 以尊重、耐心和建设性的方式交流
- 关注问题本身，而不是针对个人
- 接纳不同经验背景与技术水平的参与者
- 在反馈中提供上下文、证据和可执行建议
- 尊重他人的时间、劳动成果和创作边界
- 在讨论分歧时，优先寻求澄清与共识

## 不被接受的行为

- 人身攻击、羞辱、嘲讽、恶意挑衅
- 歧视、骚扰、威胁、跟踪或令人不适的言行
- 无意义刷屏、引战、钓鱼、恶意歪曲他人观点
- 公开泄露他人的隐私、密钥、未公开数据或安全细节
- 明知会破坏项目协作秩序仍持续进行的行为

## 适用范围

本准则适用于：

- 仓库中的代码、文档、Issue、Pull Request、Discussion
- 以本项目名义开展的聊天、协作或社区活动

## 执行方式

维护者有权根据具体情况采取以下措施：

- 提醒并要求停止不当行为
- 编辑、隐藏或删除不合适的内容
- 关闭相关 Issue / PR / Discussion
- 暂时或永久限制相关参与者的互动权限

## 反馈与处理

如果你遇到了不当行为，可以联系仓库维护者处理。若不适合在公开场合说明，请通过仓库所有者主页联系维护者，并尽量提供：

- 事件发生的位置
- 相关链接或截图
- 简要经过
- 你希望得到的处理方式

维护者会尽量审慎、及时地处理，并在必要时保护反馈人的隐私。

感谢每一位愿意让协作环境更友善、更专业的参与者。
